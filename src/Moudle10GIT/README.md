# Git Hands-On Learning (HOL) Exercises

This directory contains the structure and assets for the **Git Hands-On Learning (HOL)** exercises.

---

## 📁 Modules

- **`1. Git-HOL`**: Basic Git Commands (init, status, add, commit, log)
- **`2. Git-HOL`**: Branching & Merging (branch, checkout, merge, rebase)
- **`3. Git-HOL`**: Conflict Resolution & Stashing (stash, resolve merge conflicts)
- **`4. Git-HOL`**: Working with Remote Repositories (clone, fetch, pull, push, remotes)
- **`5. Git-HOL`**: Advanced Git Workflows & Tagging (tags, cherry-pick, reset, revert)
